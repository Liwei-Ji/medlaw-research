# medlaw-research

> 醫藥法規判決檢索與分析

以司法院裁判書為資料源,對**醫藥五法**(藥事法、醫療器材管理法、醫療法、醫師法、藥師法)
近年判決做的**檢索與分析**專案。裁判日期涵蓋 **民國 112–115 年**(115 年至 7 月),
去重後 **14,031 筆**;資料快照 **2026-07-07**。

---

## 檢索系統

RAG 是「檢索+用LLM生成答案」。專案只做檢索、沒有生成:使用者的問題經過查詢解析與索引比對後,答案由查表與模板組出,全程不呼叫語言模型。這樣做是為了可離線、零成本、結果可稽核。

換句話說,它是一套資訊檢索(IR)系統,外觀做成對話介面;與 RAG 的差別在於少了「生成」那一層。

## 專案結構

```
medlaw-research/
├── README.md            專案定位與總覽
├── crosstab.html        法院 × 年度 × 類別 互動熱力圖
├── medlaw-qa/           檢索問答對話框(App;兩層檢索架構見其 README)
├── data/                資料成品(索引 / 結構化 / 全文 / 統計)
├── pipeline/            產生上述資料的全部程式(見其 README)
└── docs/                延伸文件(學習筆記)
```

## 延伸文件

- [`medlaw-qa/README.md`](medlaw-qa/README.md) — 問答檢索架構(A 快搜 / 全文深搜)、檔案、部署。
- [`pipeline/README.md`](pipeline/README.md) — 資料爬取 → 建索引 → 下載全文 → 產生視覺化。
- [`docs/NOTES.md`](docs/NOTES.md) — 學習筆記(IR、倒排索引、中文斷詞…)與系統取捨(全文DB體積之旅)。
- [`docs/USAGE.md`](docs/USAGE.md) — 使用與部署、重建資料。

## 資料來源與使用聲明

- **來源**:司法院裁判書查詢系統(https://judgment.judicial.gov.tw/FJUD/default.aspx),為公開資料。每筆均附回原始裁判書連結,內容一律以司法院原文為準。
- **時效**:涵蓋裁判日期 112–115 年(115年僅至7月)
- **解析限制**:法院、案號、裁判日期與全文為系統實際資料;由全文以規則擷取的欄位(適用條號、刑度、緩刑、罰金/沒收、承審法官)可能有缺漏或誤差,金額類尤其不完整。
- **用途**:僅供研究與檢索,非法律意見;引用前請以原始裁判書核對。
